Tile editor help

Palette panel: One page of the palette (16 entries) out of 16 pages
(256 entries) is displayed. Cycle through the pages with the Up and Dn
buttons, enter a page directly in the edit control between them.

Editor panel: Currently only edits tiles or sprites with a height or
width of 8 or 16, but will accept any of the "Bpp" values (1, 2, 4, or 8.)

Click on a tile to select it for editing, then use left click to set a pixel
to the set color or right click to clear a pixel to 0. The following hot
keys will work in the editor panel:

d: "dropper" Sets the current color when you hold down "d" and left click on
any pixel in the editor panel, it doesn not have to be selected.
